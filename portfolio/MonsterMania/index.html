Monster Mania

December 2017 - January 2018
position: programmer

overview


Monster Mania is an interesting competitive game made by Unreal Engine. The main gameplay is to chase the ball in a small planet against another player. Take the ball to your own goal can get one point. And your opponent can make the ball drop off by knock into you.

what i did


Monster Mania a local multiplayer game. it is implemented by split screen. Players can use both keyboard and controller to play the game. As an engineer, I Use blueprint and C++ to implemented the local multiplayer gameplay, Calculated and Simulated the physics that happens in a sphere environment. This project is still in developing, we plan to Develop level manager and player manager to implement level and player selection function, and Deploy server to implement online multiplayer function.
