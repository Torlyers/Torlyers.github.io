Dead Diary

April 2017 - June 2017
position: Programmer & Designer & Artist


overview

This is a Location service based mobile game. I developed it in my spare time. My inspiration came from Ingress. The background of the game is set as the world of waste after the nuclear war. The core gameplay is to collect items in this world, build their homes, fight with the enemies in order to survive as many days in the game as possible.

what i did


I used Baidu Map SDK and Android Studio to develop this game. Essentially, it is an Android application. I implemented location service and gameplay logic. Use static file to save game data and refreshing game state on avtivity layout. Besides, I designed stylish icons and the map, trying to make it more like a "game" visually.
