Elecforce

Jabuary 2018(GGJ 2018)
position: Programmer & Technical Artist

overview

This is a multiplayer arena game made by Unity, which wins the best doodle award in the event. The theme of GGj 2018 is transmission. So we use four electrons as main characters. The core gameplay is to collect electric energy from the scene and others. Every player need to avoid traps in arena and make use of traps in the environment to eliminate others. The last electron will be the winner

what i did

As an engineer, I implemented player's movement logic and physics system. There are many kinds of traps in the arena like black hole, laser tower, and cannon so that the physical performance is different. Besides, I developed the UI of the game.

  • Implemented the main game loop logic
  • Implemented collision detection between players and walls.
  • Implemented all game UI canvas
  • Made particle effects and used post-processing to improve visual performance
