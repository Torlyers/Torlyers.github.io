Cavecity

July 2016
position: programmer & designer

A scene roaming project based on virtual reality. We built a gloomy city in the mountain. Users can walk, jump and fly in the scene. We set some aim points to make sure users can explore the whole scene. We try to create a bleak atmosphere for users.

The project can be run on HTC Vive and users must install Steam VR first. Users use gamepads to move and turn their heads to select direction.

How to Play: Touch the touchpad to move. Press trigger once to jump. Press trigger for a long time to fly. When flying, turn head to choose direction and use touchpad to fly left and right or up and down.
